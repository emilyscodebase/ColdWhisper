# ColdWhisper

ColdWhisper is an AI-assisted tool designed to support developers' emotional well-being and productivity during coding sessions.

## Concept
The idea was developed during Q-Hackathon 2025, where it ranked 4th place. ColdWhisper explores how AI can detect frustration and provide real-time support to developers.

## Current Prototype
- Detects frustration-related input
- Responds with supportive feedback
- Built using Python

## Vision
To evolve into a system that uses voice or facial signals to detect emotional states and provide intelligent interventions.

This repository also includes a personal journey and project background.

## Status
Work in progress — actively developing features and improving system intelligence.
